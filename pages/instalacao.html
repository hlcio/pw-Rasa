Como aprender RASA

Olá, event.tagName()

Rasa é um framework open source de Machine Learning para a criação de assistentes de IA e chatbots. Geralmente, você não precisa de nenhuma experiência em linguagem de programação para trabalhar com Rasa. Embora exista o “Rasa Action Server”, onde você precisa escrever código em Python, que é usado principalmente para acionar ações externas, como chamar API do Google ou API REST, etc.
