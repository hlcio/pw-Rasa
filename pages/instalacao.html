Instalação

1. Configuração do ambiente Python #

Verifique se o seu ambiente Python já está configurado:
python3 --version
pip3 --version
Se esses pacotes já estiverem instalados, esses comandos devem exibir os números da versão de cada etapa e você pode pular para a próxima etapa.

2. Configuração do ambiente virtual

Esta etapa é opcional, mas é altamente recomendável isolar projetos Python usando ambientes virtuais. Ferramentas como virtualenv e virtualenvwrapper fornecem ambientes Python isolados, que são mais limpos do que a instalação de pacotes em todo o sistema (pois evitam conflitos de dependência). Eles também permitem que você instale pacotes sem privilégios de root.

3. Instale o Rasa Open Source

Primeiro, certifique-se de que sua versão do pip está atualizada:
pip3 install -U pip
Para instalar o Rasa Open Source:
pip3 install rasa
